# Simple PostgreSQL Backend

A beginner-friendly FastAPI app that connects to PostgreSQL.

## What's Inside

- **FastAPI** - Simple web framework
- **PostgreSQL** - Database  
- **Docker** - Easy setup



## How to Run

1. **Start everything:**
   ```bash
   docker-compose up --build
   ```

2. **Check if it works:**
   - Open `http://localhost:8001`
   - Should say "Database is connected "

3. **Stop everything:**
   ```bash
   docker-compose down
   ```
